Features

  • Profile Image Display

    Shows uploaded student profile pictures in circular containers

  • Letter Avatar Fallback

    Displays first letter when no image is available (e.g., "P" for Paras)

  • Multiple Sizes

    4 size options: sm (32px), md (48px), lg (64px), xl (96px)

  • Google-Style Colors

    Uses Google's color palette for consistent, recognizable letters

  • Optional Name Display

    Can show user's name and email next to avatar

  • Responsive Design

    Works perfectly on all screen sizes and devices
